# Bakers by the Dozen

Bakers by the Dozen is a mobile baking and recipe-sharing application designed to help users discover, create, save, and manage recipes in a centralized and user-friendly environment. The application focuses on simplifying recipe management while providing ingredient scaling and unit conversion tools for baking enthusiasts and professionals alike.

The platform combines recipe organization with social interaction, allowing users to explore recipes shared by others, save favorites, and interact with baking content through a modern mobile interface.

## Features

- Create and save custom recipes
- Browse and explore recipe feeds
- View detailed recipe ingredients and instructions
- Ingredient scaling based on serving size
- Unit conversion between supported measurements
- Public and private recipe settings
- Recipe sharing functionality
- User-friendly mobile interface

## Technologies Used

- Kotlin
- Android Studio
- Jetpack Compose
- Gradle
- GitHub

## How to Run the Project

1. Clone the repository
2. Open the project in Android Studio
3. Allow Gradle to sync all dependencies
4. Run the application using an Android emulator or Android device

## Project Purpose

Bakers by the Dozen was developed as a collaborative mobile application focused on recipe management, ingredient scaling, measurement conversion, and social recipe sharing. The project emphasizes usability, accessibility, and community interaction within a baking-focused platform.
